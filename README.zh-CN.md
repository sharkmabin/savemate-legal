# SaveMate 法律页面

SaveMate 的公开法律与政策页面，托管于 GitHub Pages。

## 项目内容

- 多语言页面：英文（默认）、简体中文、繁体中文、日文、韩文、泰文
- 三种页面类型：首页、隐私政策、版权/DMCA
- 统一的 `styles.css` 样式文件

## 页面地址

隐私政策（Google Play 使用英文默认地址）：
`https://sharkmabin.github.io/savemate-legal/privacy-policy.html`

各语言隐私政策：
- 简体中文：`privacy-policy.zh-CN.html`
- 繁体中文：`privacy-policy.zh-TW.html`
- 日文：`privacy-policy.ja.html`
- 韩文：`privacy-policy.ko.html`
- 泰文：`privacy-policy.th.html`

## 技术栈

- HTML / CSS
- GitHub Pages

## 说明

如需启用 GitHub Pages，在仓库 Settings > Pages 中设置 Source 为 `Deploy from a branch`，选择 `main` 分支、`/root` 目录即可。
