## Enable GitHub Pages

In GitHub repository settings:

1. Open Settings.
2. Open Pages.
3. Set Source to `Deploy from a branch`.
4. Set Branch to `main`.
5. Set Folder to `/root`.
6. Save.

After GitHub Pages finishes deployment, use the Privacy Policy URL above in Google Play Console.
